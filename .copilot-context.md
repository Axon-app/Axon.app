# GitHub Copilot - Configuración de proyecto

# Este archivo ayuda a Copilot a entender mejor el contexto del proyecto

## Información del proyecto

- **Nombre**: Axon.app
- **Tipo**: Sitio web corporativo
- **Framework**: React 19+ con Vite 6.x
- **Estilos**: Tailwind CSS 4.x
- **Formularios**: Sistema unificado sin dependencias externas

## Stack tecnológico

- React (funcional components + hooks)
- Vite (build tool)
- Tailwind CSS (utility-first)
- JavaScript ES6+ (no TypeScript)
- Canvas API (para imágenes)

## Patrones de código preferidos

- Functional components únicamente
- React hooks (useState, useEffect, useMemo, useCallback)
- Arrow functions
- Template literals
- Destructuring
- Modules ES6 (import/export)

## Estructura de archivos

```
src/
├── components/
│   ├── forms/           # Formularios
│   ├── modals/          # Modales
│   └── ui/              # Componentes UI
├── data/                # Datos estáticos
├── hooks/               # Custom hooks
└── services/            # Servicios
```

## Estilo de código

- Usar dobles comillas para strings
- Punto y coma al final
- Indentación con 2 espacios
- Nombres descriptivos en camelCase
- Comentarios en español

## Convenciones específicas

- Prefijo "Enhanced" para componentes mejorados
- Hooks personalizados con prefijo "use"
- Constantes en UPPER_CASE
- Archivos de componentes en PascalCase

Este contexto ayuda a Copilot a generar código más preciso y consistente.
