### Troubleshooting

#### If Preview Doesn't Show

1. **Clear cache**: Try incognito/private browsing
2. **Wait a moment**: Sometimes takes time to load
3. **Check URL**: Make sure it's the correct link
4. **Platform delays**: Some platforms cache slowly

#### Platform-Specific Issues

##### Facebook Issues

- Use [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- Click "Scrape Again" to refresh preview
- Check for blocked content

##### Twitter Issues

- Use [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- Verify all meta tags are present
- Check image accessibility

##### LinkedIn Issues

- Try sharing again after a few minutes
- Use the LinkedIn Post Inspector
- Check privacy settings
